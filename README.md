# Habit Tracker MVP

## What It Does
The Habit Tracker MVP is a simple web application that allows users to create, track, and manage daily habits. Users can add habits, mark them as completed, and monitor their progress in real time.

## How to Use
1. Open the application in your web browser
2. Type a habit into the input field
3. Click the "Add" button
4. Click the ✔ button to mark a habit as complete
5. Click the ✖ button to delete a habit

## Features Implemented
- Add new habits
- Mark habits as complete/incomplete
- Delete habits
- Real-time progress tracking (completed vs total)
- Input validation to prevent empty habits
- Clean and simple user interface
